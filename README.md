# Liquid 27
The new iOS 27 icons brought to older iOS devices. Icons are from [Apple Design Resources](https://developer.apple.com/design/resources/)

Liquid 27 is a theme that ports/brings the new iOS 27 icons to older idevices. With 2 modes: Default(Light) and Dark.

# Development
this is still in development so it might not be stable. I only tested this on an iPhone 8 running the Darksword exploit.
